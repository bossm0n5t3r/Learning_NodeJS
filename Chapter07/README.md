# MySQL

## What is a database?
- **데이터베이스**는 관련성을 가지며 중복이 없는 데이터들의 집합
- 이러한 데이터베이스를 관리하는 시스템을 **DBMS**(데이터베이스 관리 시스템)라고 부름
- 서버에 데이터베이스를 올리면 여러 사람이 동시에 사용할 수 있음
- **RDBMS**
  - 관계형 DBMS
  - Oracle, MySQL, MSSQL 등
